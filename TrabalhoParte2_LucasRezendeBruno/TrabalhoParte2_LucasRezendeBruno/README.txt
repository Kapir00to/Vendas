OBS: Para que o programa funcione, deve-se alterar o atributo path ou path2, do tipo Path, das classes JCliente, JProduto, JVenda e NovaVenda para o diretório no qual deseja colocar o arquivo relativo às classes Cliente, Produto e Venda.
Cada classe tem um arquivo txt diferente que armazena a lista de objetos da classe específica.
Para executar o código como um todo, execute a classe Visao.
